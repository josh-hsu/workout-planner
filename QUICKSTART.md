# 快速開始指南

## 安裝步驟

1. **安裝依賴套件**:
   ```bash
   pip install -r requirements.txt
   ```

2. **準備工作目錄**:
   - 在專案目錄下建立課程種類資料夾（例如: `BodyCombat`, `BodyPump`）
   - 將影片檔案 (.mp4) 放入對應的資料夾中

   目錄結構範例:
   ```
   workout-planner/
   ├── BodyCombat/
   │   ├── BC64.mp4
   │   ├── BC65.mp4
   │   └── ...
   ├── BodyPump/
   │   ├── BP100.mp4
   │   └── ...
   └── ...
   ```

3. **啟動應用程式**:
   ```bash
   python main.py
   ```

## 基本使用流程

### 步驟 1: 建立分段描述檔

1. 啟動應用程式，點擊「建立分段描述檔」
2. 選擇要處理的影片檔案
3. 播放影片，找到每個片段的開始位置，點擊「標記開始」
4. 找到片段的結束位置，點擊「標記結束」
5. 在彈出的對話框中輸入:
   - 序號（自動填入，可手動調整）
   - 歌名（可選）
   - 訓練名稱（可選）
6. 重複步驟 3-5，標記所有片段
7. 完成後點擊「匯出描述檔」

### 步驟 2: 建立播放清單

1. 點擊主畫面的「建立課程播放清單」
2. 選擇課程種類（會顯示所有有描述檔的影片和片段）
3. 雙擊想要加入的片段，會出現在右側的「已選分段清單」
4. 繼續選擇更多片段，可以跨不同影片選擇
5. 檢查右上角的總時長
6. 點擊「匯出播放清單」並輸入名稱
7. 完成！播放清單會儲存在 `playlists/` 目錄

### 步驟 3: 播放

使用 VLC Media Player 開啟 `playlists/` 目錄下的 .xspf 檔案即可播放。

## 提示與技巧

### 建立分段描述檔時:
- 可以使用播放/暫停按鈕精確定位
- 拖動進度條快速跳轉到不同位置
- 序號會自動按時間排序，不用擔心標記順序
- 可以隨時編輯或刪除已標記的片段

### 建立播放清單時:
- 雙擊片段即可加入清單
- 同一片段可以重複加入
- 可以從不同影片選擇片段組合
- 右側顯示即時的總時長

### 最愛功能:
- 在課程分段資訊中，★ 表示已加入最愛，☆ 表示未加入
- 目前最愛標記會記錄在配置檔中（未來可能新增篩選功能）

## 常見問題

### Q: 為什麼影片播放器無法顯示影片？
A: 請確認已安裝 opencv-python 和 Pillow:
```bash
pip install opencv-python Pillow
```

### Q: 可以使用其他格式的影片嗎？
A: 目前僅支援 .mp4 格式

### Q: 播放清單可以在其他播放器使用嗎？
A: XSPF 是標準格式，但片段播放功能需要支援 VLC 擴展語法的播放器

### Q: 如何備份配置和播放清單？
A: 複製以下檔案即可:
- `.workout-planner` (配置檔案)
- `playlists/` 目錄（所有播放清單）
- `*/*.json` (所有分段描述檔)
